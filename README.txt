CREATIVE DEPARTMENT — DEPLOYMENT FOLDER
========================================

Ready to upload to any web host (Netlify, Vercel, Hostinger,
cPanel, GoDaddy, SiteGround, etc.). No build step needed.

WHAT'S INSIDE
-------------
index.html   →  The Creative Department website (editorial one-pager:
                hero, services, the scroll journey, work, about,
                love letters, Instagram, DM contact card).
                Bilingual (ES / EN), SEO tags and favicon included.
uploads/     →  Photos used by the services & portfolio sections.
                Keep this folder next to index.html.

HOW TO DEPLOY
-------------
1. Upload index.html AND the uploads folder to your host's
   public root (often called public_html, www, or htdocs).
2. Your site goes live at yourdomain.com

Drag-and-drop deploy (easiest):
   Go to app.netlify.com/drop and drop this folder in.
